# Currency Converter App 💱

A responsive, modern currency converter built with **React**, **Tailwind CSS**, and **Framer Motion**.  
Convert currencies in real-time using the **ExchangeRate-API**, with auto-convert, swap animations, conversion history, and dark mode.

## Features

- Real-time currency conversion
- Auto-convert while typing
- Live exchange rate preview
- Swap currencies with smooth rotation animation 🔄
- Conversion history log
- Dark mode toggle 🌙 / ☀️
- Responsive design (desktop, tablet, mobile)
- Smooth Framer Motion animations

## Tech Stack

- React (v18+)
- Tailwind CSS
- Framer Motion
- ExchangeRate-API
